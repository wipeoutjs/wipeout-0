New global functionality

Global functionality can easily be added to wipeout native classes because of their prototypical inheritance structure.

Say you every view, contentContral, if and itemsControl to have a unique id. This would involve adding functionality to the class as all other classes inherit from it:

// Use a closure so that id can not be corrupted (function () { var id = 0; // add functionality to the view prototype wo.view.prototype.uniqueId = function () { // lazily generate an id if(this._uniqueId === undefined) { id++; this._uniqueId = id; } return this._uniqueId; }; }());

Now every class which inherits from wo.view has a uniqueId function

Using custom items control items

With a custom itemsControl you can control .

var myItemsControl = wo.itemsControl.extend({ constructor: function myItemsControl() { this._super(); }, createItem: function(model) { var item = new myViewModelType(); item.model(model); item.templateId(this.itemTemplateId()); return item; } });

With a regular items control, all items are plain wo.view objects. If you override the createItem method you can alter this.

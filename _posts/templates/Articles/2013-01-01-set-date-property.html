Set a date property

The date parser uses new Date(value) to parse dates. You can override this by overriding the date parser itself.

<wo.contentControl> <myVal constructor="date"> 2001/01/01 </myVal> <template> Binding to element value: <span data-bind="text: myVal"></span> </template> </wo.contentControl> <wo.view templateId="'set-date-property-script-1'" />
